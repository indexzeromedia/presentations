THE CENTRAL PROCESSING UNIT (CPU)

Bill Gates: "The CPU is that little something that makes the computer do stuff."

HARD DISK DRIVE (HDD)

Hard Drives are where the permanent memory is stored, or better yet the persistant memory.

RANDOM ACCESS MEMORY (RAM)

The RAM is where the temporary memory is stored, or the memory that gets cleared when it is not needed anymore.
