CRUCIAL RULES TO UNDERSTAND BEFORE YOU START PROGRAMMING
Why do we need to talk about this?
  • Everybody is talking about programming these days
  • People think that it is very hard to start with it
  • There are no "hardcoded" rules to follow
  • Overwhelming amount of resources
As a self-taught developer myself, I really started learning after I started following these rules.
1. THE PROGRAMMING COMMUNITY IS LARGE, SO GET INVOLVED.
  • Confidence
  • Motivation
  • Knowledge
  • Respect in the community
You can achieve this in two ways.
  • Face-to-face conversation with other programmers
  • Using social networks/programming communities on the Internet
2. GOOGLE IS YOUR FRIEND, USE IT.

Not a single person can remember all those lines of code, therefore, using Google is a must in programming.

3. STAY FOCUSED ON WHAT YOU'RE DOING.
  • Nobody is immune to distractions
  • Multitasking is a myth, you're switchtasking
  • Enjoy learning a new skill
4. PROGRAMMING IS A SINUSOIDAL PROCESS, IT IS NEVER LINEAR.

There will always be ups and downs while learning, even after it. This applies to everything.

  • The linear way of programming is a mistake
  • Concentration is needed the most at the beginning
  • Listen to instructions
